🔐 Network Scanning and Traffic Analysis

📌 Project Overview

This project focuses on analyzing network behavior and identifying potential vulnerabilities using various cybersecurity tools. It demonstrates practical implementation of network scanning, port analysis, traffic monitoring, request interception, and communication between systems.

The project was carried out as part of coursework at VIT Bhopal University in the domain of Cybersecurity.

---

🎯 Objectives

- Identify active devices in a network
- Scan open ports and detect services
- Analyze real-time network traffic
- Intercept HTTP requests
- Simulate communication between systems

---

🛠️ Tools Used

- Angry IP Scanner – Network scanning
- Nmap – Port scanning and service detection
- Wireshark – Packet capture and traffic analysis
- Burp Suite – HTTP request interception
- Netcat – Network communication

---

⚙️ Methodology

The project follows a structured workflow:

1. Identify system IP address
2. Scan network for active hosts
3. Perform port scanning
4. Capture network traffic
5. Intercept web requests
6. Establish communication

All activities were performed in a controlled and authorized environment for educational purposes only.

---

📊 Key Features

- Network discovery using IP scanning
- Identification of open ports and services
- Real-time packet analysis
- Web request interception using proxy
- Communication simulation between systems
